CSS Properties Reference

Each element on the page can have many style properties:

Name Usage Important info/values Examples
font-family Specifies the typeface of text
  • The font must be installed on the user's computer or linked as a web font
  • When using a multi-word font name - use "..."
 font-family:"Courier New";
font-size Specifies the size of the font
  • Can be specified in px, %, ems
  • Default size of text in browsers is 16px
 font-size: 12px;
font size: 75% (=12px)
font-weight Controlls how bold or thin text appears
  • Can be represented with numbers or words
  • Font-weight:400 is considered normal
  • Font-weight:700 is considered bold
 font-weight: bold;
font-weight: normal;
font-style Changes appearance of text
  • It can take 3 values: normal, italic, oblique
  • Oblique is a regular version of type put on an angle
font-style: italic;
text-transform Changest the case of text Takes values: uppercase, lowercase, capitalize text-transform: uppercase;
line-height Sets the height of an entire line of tex Its value is best given in ems,so it is relative to the size of text line-height: 1.4em;
letter-spacing Sets the horizontal spacing between each letter in an element It is useful to increase it on uppercase text to make it easier to read letter-spacing: 0.2em;
word-spacing Controls the gap between words
  • Its value should be given in ems
  • It is added on top of the default value specified by the font
 word-spacing: 1em;
text-align Controls the alignment of text
  • Takes values: left, right, center, justify
  • Aligns text to the element that holds it (to its parent).
 text-align:center;
color Styles an element’s foreground color
  • color names: 147 predefined color names
  • rgb: numbers between 0 and 255 for each of the colors
  • hex codes: values for colors in hexadecimal code
  • hsl: hue(color itself(0-360)), saturation(intensity-%), lightness(50% normal,0% darkest,100% lightest)
  • hsla: hsl plus another value- alpha, which is the opacity(0-1)
 color:rgb(100, 100, 100);
color:#ee3e80;
color:hsl(240,50%,45%);
color:hsla(240,30%,90%,0.5);
background-color Styles an element’s background color CSS treats each HTML element as if it appears in a box, and'background-color'sets the color of the background for that box. background-color: aqua;
opacity Controls the transparency of an element Its value ranges from 0 (transparent) to 1 (opaque) opacity: 0.5;
background-image Sets the background image of an element An image URL should be provided url("moon.jpg") as the value of the property background-image: url("...")
border-radius Can give an element rounded curved corners At 50% image will become a circle border-radius:50%;
